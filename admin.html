Change Admin Password

Password changes are applied to the backend server via your Render.com environment variables. Update SUPER_ADMIN_PWD there to change your login password.

How to change your admin password

1. Log in to dashboard.render.com
2. Select your Web Service → Environment
3. Edit the SUPER_ADMIN_PWD variable
4. Click Save Changes — service will restart automatically
